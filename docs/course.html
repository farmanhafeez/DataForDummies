Course data

Documentation for the Random data Generator API

How to use

Random Course Generator allows you to fetch up to 2000 generated course in one request. Below mentioned URL is used to request course data.

https://www.example.com/api/course/

Sample data

The API will provide you the response with a JSON format.

{
    "course": [
        {
            "course_id": "CRS-17146",
            "course_abr": "B.Tech",
            "course_name": "B.Tech. Chemical Engineering",
            "department_abr": "UG",
            "department_name": "Undergraduate"
        }
    ]
}

Results

If you want only limited amount of course data then you can mention result parameter as shown below.

https://www.example.com/api/course/?result=500

Department

You can specify whether you would like to have only specific department course generated by adding the department parameter to your request. Valid values for the department parameter are "UG" or "PG"

https://www.example.com/api/course/?department=UG

Note:

Our API offers a maximum of 2000 records for a API request. If you request more than 2000 records then we will return an error response. Best way to generate more records is to use our Data Generation Tool to generate upto 5000 records for a dataset.
